# MuseScore Plugins

This repository is a collection of useful plugins for MuseScore Studio 4

## Content

- [parallelIntervalsChecker](/parallelIntervalsChecker/)
- [choraleRulesChecker](/choraleRulesChecker/) — Bach chorale rules checker (parallels + harmony/voice-leading rules, each toggleable and with a bibliographic reference; bilingual EN/ES)
